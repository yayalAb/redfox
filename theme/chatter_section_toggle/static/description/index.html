How to Use the Chatter Toggle

With our module installed, every form now includes two handy buttons placed below the main form area:

Show Chatter (<<<)

Click this button to instantly display the Chatter panel, expanding the form to accommodate the conversation thread. The form width adjusts to 1534px for better visibility.

Hide Chatter (>>>)

Click this button to hide the Chatter panel, restoring the form’s original width. Great for a distraction-free view when you don't need the communication panel open.

The toggle buttons automatically appear only if the form contains a Chatter panel. Enjoy seamless control of your workspace—no manual configuration needed.
